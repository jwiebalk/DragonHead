Using a [Elegoo Nano](https://www.amazon.com/Arduino-Elegoo-ATmega328P-without-compatible/dp/B0713XK923) 
[Servos](https://www.amazon.com/TowerPro-SG90-Mini-Servo-Accessories/dp/B001CFUBN8)
